# Monokai Pro (Community Edition) for [Xfce Terminal](https://docs.xfce.org/apps/xfce4-terminal/start)

## Installation instructions

Clone from Github:

```zsh
git clone https://github.com/monokai-pro/xfce4-terminal.git
```

### Activating theme

1. Copy `monokai-pro.theme` to `~/.local/share/xfce4/terminal/colorschemes/`
2. Activate the theme, in the color settings, in the presets list
3. Enjoy!
